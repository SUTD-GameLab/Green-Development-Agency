Creating Better/Additional Asset Generators

Currently in the game, there are a few ScriptableObjects that can be used to create the game assets. For example, ProductGenerators generate Products, UserArchetypes generate UserGroups, and ProposalSequenceItem generate Proposals. When numerical values need to be generated, a normal distribution is used. This is might result in some products that are bad/good across all fields, which may not be good balancing for the game.

These generators can be extended to provide more functionality for the game designer. All that has to be done is to implement a class that inherits these generators, or to change the generators directly.

Ideas for better generators:

  • Different statistical distributions can be used in the generation of product and user group details
  • A point allocation system can be implemented so that products that perform better in some aspects will be balanced to be worse in other aspects (instead of being independent of each other and random)
  • A proposal generation system that is more flexible in the level of scriptedness/randomness
